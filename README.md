# live-msg-selector
抖音直播评论抓取python脚本

提供http接口，传入直播地址参数即可全自动化获取直播中的评论和观众进场等消息

## 消息类型包括：
- 评论
- 到访
- 礼物
- 关注
- 点赞
- 直播间统计
- 直播间状态变更
- 粉丝团消息

## 客户端支持
- redis消息队列
- websocket长连接
  - 支持一个客户端订阅自定义数量的直播消息，一个，多个或所有，详情请参看下方演示视频
